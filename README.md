## 🛠 Setup

To add the package to the Flutter project, add the following lines to your `pubspec.yaml` file:

```yaml
dependencies:
  flutter_penta_core:
    git: https://github.com/ANILSRGT/flutter_penta_core.git
```

Then run the following command to include the package in your project:

```console
$ flutter clean
$ flutter pub get
```
